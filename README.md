## Tim „Algo_4”

Aplikacija nastala na devetom [SICEF Hakatonu](https://hakaton.sicef.info/), 24-časovnom takmičenju u programiranju održanom od 3. do 5. novembra 2023. godine na Elektronskom fakultetu u Nišu.
